Reversing CRC32

  1. Generate a reference table (may require several GBs of storage).
from crc32_table import *
gen_crctable()
  1. Infer user ID based on the reference table
from crc32_reverse import *
find_uid(uid_crc32)

About 8% of CRC32-UIDs are not unique and no way to fix this.
